# Auralis

Frontend and backend are now separated in the repo.

## Layout

- `frontend/` - React app, Vite config, frontend env files
- `backend/` - Express + MongoDB API

## Run

Frontend:

```bash
npm run dev
```

Backend:

```bash
cd backend
npm run dev
```

## Notes

- Frontend source lives in `frontend/src/`
- Backend source lives in `backend/src/`
- Put frontend env values in `frontend/.env.local`
