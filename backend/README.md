# Backend

Standalone Node.js + Express + MongoDB API for Auralis.

## Start

```bash
cd backend
npm install
npm run dev
```

## Structure

- `src/server.js` app entry
- `src/app.js` express app setup
- `src/config/` database config
- `src/models/` MongoDB models
- `src/routes/` CRUD routes
- `src/scripts/seed.js` sample dataset loader
